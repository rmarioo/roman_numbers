# Roman numbers kata

The purpouse is to translate Arabic to Roman numbers  
  
Examples:  
`2 -> II`  
`9 -> IX`

Different programming styles in different languages are developed:

**Kotlin**

* imperative solution 
* Functional solution
* Functional solution with function composition

**Scala** 

* Functional recursive solution
